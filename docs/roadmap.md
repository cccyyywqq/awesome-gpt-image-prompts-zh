# Roadmap

## 近期

- 补齐结构化数据，让 JSON 与 Markdown 案例数量逐步一致。
- 为每个分类加入更多中国本土场景案例。
- 收集不同模型对中文文字、商品包装和人像细节的表现差异。
- 增加生成结果图的推荐命名规范和目录结构。

## 中期

- 建立一个可浏览、可搜索、可复制的静态网站。
- 增加提示词评分字段，例如文字可控性、商业可用性、复用难度。
- 为常见行业提供专题集合，例如餐饮、教育、文旅、消费电子。
- 增加自动化脚本，从 JSON 生成分类 Markdown 索引。

## 长期

- 形成中文图像提示词评测集。
- 建立社区贡献的模型表现对比表。
- 提供轻量 CLI 或浏览器扩展，便于快速检索和复制提示词。
