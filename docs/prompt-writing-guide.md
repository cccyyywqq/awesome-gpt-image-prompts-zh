# 中文图像提示词写作指南

这份指南用于统一项目里的提示词质量。目标不是把句子写长，而是让模型更容易理解画面、让贡献者更容易复用和改写。

## 基本结构

推荐顺序：

```text
用途与比例 -> 主体 -> 场景 -> 构图 -> 风格 -> 光线 -> 色彩 -> 材质 -> 文字 -> 限制条件
```

如果要稳定生成实图案例，建议把提示词写成“生产级 brief”，而不是只写一句画面描述。结构可以更明确：

```text
用途 -> 资产类型 -> 案例 ID -> 必须比例 -> 主视觉 brief -> 分类专属质量要求 -> 文字约束 -> 缩略图可读性 -> 禁止项
```

这种写法更适合 `gpt-image-2`：模型能先理解图片要被用在哪里，再处理主体、版式、文字和限制条件。尤其是中文海报、社媒图、商品包装和信息图，文字必须单独列为约束，不要藏在长句里。

示例：

```text
请生成一张 4:5 新品海报。主体是一瓶磨砂玻璃香水，放在浅灰石材台面中央；背景是柔和的晨光窗影，近景有少量白色花瓣。构图正中，产品占画面 60%，顶部留出标题空间。整体风格高级、真实、克制，瓶身标签文字为「晨雾香水」。避免错误文字、产品形变、过度装饰和廉价塑料感。
```

## 写作原则

- 先说明用途和比例，例如电商主图、社媒封面、海报、头像、展厅效果图。
- 主体要具体，包含颜色、材质、形状、角度和占比。
- 场景不要只写氛围，要写清背景、道具、空间关系和复杂度。
- 光线建议明确来源，例如窗边自然光、柔和棚拍光、侧逆光、夜晚霓虹混合光。
- 中文文字要短，尽量只保留标题、副标题、品牌名或日期。
- 如果画面含中文文字，单独写一行“仅包含这些文字”，并列出精确字符串。
- 每个分类补一条质量要求，例如电商强调产品轮廓，人像强调自然肤色，建筑强调空间合理。
- 限制条件直接写在同一条提示词末尾，方便复制使用。

## 常用模板

### 商业海报

```text
请生成一张 {比例} 商业海报。主体是 {产品/人物/活动}，位于画面 {位置}；背景为 {场景}，包含 {关键道具}。构图要求 {留白/信息区/主体占比}，光线为 {光线类型}，色彩为 {色彩倾向}。画面文字仅保留「{主标题}」和「{副标题}」。整体风格 {风格关键词}，避免 {限制条件}。
```

### 电商主图

```text
请生成一张 {比例} 电商主图。主体是 {产品名}，材质为 {材质}，以 {角度} 展示，产品占画面 {占比}；放置在 {台面/环境} 中，周围只有 {少量道具}。光线为柔和棚拍光，边缘高光清晰，背景干净。避免产品形变、错误文字、杂乱背景和过度装饰。
```

### 人像摄影

```text
请生成一张 {比例} 人像摄影作品。人物为 {年龄段/职业/气质}，穿 {服装}，姿态为 {姿态}，表情 {表情}；背景是 {场景}，使用 {镜头/构图}。光线为 {光线类型}，肤色自然，画面真实。避免脸部变形、过度磨皮、多余人物和低清晰度。
```

### 生产级实图案例

```text
用途：{商业海报/电商主图/人像摄影/角色设定/社媒封面/建筑效果图}
资产类型：中文图像提示词库官方示例图
案例 ID：{prompt-id}
必须比例：{比例}
主视觉 brief：{完整画面描述}
质量要求：{分类专属要求，例如产品轮廓清晰、空间合理、信息层级清楚}
文字约束：仅包含「{文字 1}」「{文字 2}」，文字要大、清楚、位置符合描述；不要添加其他文字。
缩略图要求：在 GitHub README 缩略图尺寸下也能一眼看懂主体和层级。
禁止项：{负面提示词}、水印、真实品牌标识、受版权保护角色、真实名人肖像、乱码、无关英文。
```

## 质量检查

提交前请确认：

- 读者不用上下文也能理解画面。
- 提示词不是英文提示词的机械翻译。
- 文字内容短且明确。
- 风格词服务于画面，而不是堆叠流行词。
- 限制条件与这个场景真的有关。
