# 贡献指南

谢谢你愿意贡献中文图像提示词。这个项目更看重可复用、可改写、能稳定产出结果的提示词，而不是单次偶然好看的句子。

## 提交内容

每条提示词建议包含：

- 标题：一句话说明用途。
- 分类：从现有 `cases/` 分类中选择。
- 画幅比例：例如 `1:1`、`3:4`、`16:9`。
- 正向提示词：中文为主，尽量完整描述画面。
- 限制条件：说明需要避免的问题。
- 标签：方便检索。
- 可选：生成图、模型版本、参数、失败案例。

## 写作建议

- 先写用途，再写画面，不要只堆风格词。
- 明确主体、构图、光线、材质、文字和留白位置。
- 中文文字要短，最好只保留必要标题和副标题。
- 商业图要写清产品占比、镜头角度和背景复杂度。
- 人像图要写清年龄段、气质、服装、姿态和镜头语言。
- 不提交侵犯第三方版权、商标或肖像权的内容。

## 数据格式

如果你同时更新 `data/prompts.zh-CN.json`，请保持字段符合 [schema/prompt.schema.json](schema/prompt.schema.json)。

提交前可以用 PowerShell 检查 JSON：

```powershell
Get-Content data/prompts.zh-CN.json -Raw | ConvertFrom-Json | Out-Null
```

## Pull Request

请在 PR 中说明：

- 新增或修改了哪些分类。
- 是否附带生成结果图。
- 是否验证过中文文字可读性。
- 是否存在模型版本相关的限制。
